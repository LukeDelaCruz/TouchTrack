# TouchTrack
Android Client and Python Server System that gives mobile phones cursor control of the host PC using WebSockets over WiFi. Innovation Award Winner 2018*


# Setup    
  * Install [flask-sockets](https://github.com/kennethreitz/flask-sockets)
  * Install [flask](http://flask.pocoo.org/)
  * Install [pyautogui](http://pyautogui.readthedocs.io/en/latest/install.html)
Please take NOTICE: This project was developed and tested strictly in a Windows 10 environment with a Samsung Galaxy Grand Prime

Mac OS:

`sudo pip3 install pyobjc-framework-Quartz`

`sudo pip3 install pyobjc-core`

`sudo pip3 install pyobjc`


Ubuntu:

`sudo pip3 install python3-xlib`

(Scrot is a screenshot program that PyAutoGUI uses.)
`sudo apt-get install scrot` 

`sudo apt-get install python3-tk`

`sudo apt-get install python3-dev`

* Install the app by moving the APK to a local folder in your Android device, then click the file to install (permissions have to be granted)

# Usage
1. Connect and authenticate w Wi-Fi on laptop and mobile (must be in the same subnet)

2. Run the server using:
  python server.py
  or
  python3 server.py

3. Open the app and input the server IP and security code information
generated by the server.

4. Follow server prompts on your PC/Mac.
